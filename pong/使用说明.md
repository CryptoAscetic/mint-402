# X420Mint 使用说明

## 环境设置

### 1. 激活虚拟环境
```bash
source venv/bin/activate
```

### 2. 配置私钥
编辑 `pong.py` 文件，在第14行填入你的私钥：
```python
"privateKey": "你的私钥 0x 开头",  # 请填入你的真实私钥
```

### 3. 调整配置（可选）
根据需要修改以下参数：
- `UsdcAmount`: USDC 数量（默认10）
- `threadCount`: 线程数（默认30）
- `totalMintCount`: 总铸造次数（默认100000）

## 运行程序

```bash
python pong.py
```

## 重要提醒

⚠️ **请务必注意以下风险**：
1. 项目已经超额，项目方一直不发币
2. 需要真实的 USDC 进行铸造
3. 请谨慎操作，建议先用少量金额测试
4. 私钥安全非常重要，请妥善保管
